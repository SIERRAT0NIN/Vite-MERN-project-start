# MERN Project Starter Kit

This repo was created using:
-Vite
-React
-TailwindCSS
-NextUI
-React Router
-Next.js
-Mongo DB Server

# Instructions
-Fork the repo
-Clone using SSH
cd Vite-MERN-project-start
$npm install

To run the frontend server:
$npm run dev

To run database
$npm run server

Connect your MongoDB in server/server.js
